### Resource Cost Scaling
---
Scales all piece and crafting resource costs using a single configuration value.

Rounds toward nearest integer, midpoints round to even numbers.

```
2 x 1.25 = 2.5 -> 2
2 x 1.3 = 2.6 -> 3
```
```
2 x 1.7 = 3.4 -> 3
2 x 1.75 = 3.5 -> 4
```
