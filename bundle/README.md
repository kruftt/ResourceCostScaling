### Resource Cost Scaling
---
Scales all piece and crafting resource costs using a single configuration value.

Rounds up unless in the bottom 0.1 of a cost value. 
For example:
```
2 x 1.10 = 2.2 -> 3
2 x 1.05 = 2.1 -> 2
...
2 x 0.60 = 1.2 -> 2
2 x 0.55 = 1.1 -> 1
```
